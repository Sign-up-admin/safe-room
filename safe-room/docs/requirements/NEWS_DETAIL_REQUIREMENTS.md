---# 新闻详情页需求文档（News Detail v1.0）

> 版本：v1.0  
> 更新日期：2025-11-15  
> 适用路由：`/index/newsDetail?id={newsId}`  
> 关联模块：`moduleKey = 'news'`

---

## 0. 设计关键词

沉浸阅读 · 排版优化 · 互动延伸 · 分享传播 · 推荐联动

---

## 1. 页面结构

| 顺序 | 区块 | 目的 |
| --- | --- | --- |
| ① | 标题 & 元信息 | 展示标题、类型、发布时间、阅读数 |
| ② | Hero 媒体 | 大图/视频，营造氛围 |
| ③ | 正文内容 | 富文本排版，含图文、引用、列表 |
| ④ | 重点摘录 | 高亮关键句、数据 |
| ⑤ | 分享 & 收藏 | 提供分享、点赞、收藏操作 |
| ⑥ | 相关推荐 | 推荐相关资讯、课程或活动 |
| ⑦ | 评论区（可选） | 用户互动 |

---

## 2. 视觉规范

- 标题使用金色描边 + 字距拉伸
- Hero 媒体支持渐入动画，叠加粒子
- 正文排版使用 72 列网格，首字母放大
- 引用块采用发光左边框

---

## 3. 功能模块

### 3.1 元信息

- 显示类型、作者、发布时间、阅读量
- 阅读量实时更新（进入页面触发 `/news/addClick`）

### 3.2 富文本内容

- 支持图片、视频、嵌入组件（如训练数据）
- 图片提供放大/灯箱
- 代码块/引用样式对齐品牌风格

### 3.3 重点摘录

- 自动提取关键句或由后台标记
- 展示在侧栏或插入正文

### 3.4 分享 & 收藏

- 分享平台：微信、微博、复制链接
- 收藏：调用 `/storeup/save`（类型 news）
- 点赞：可选，记录用户行为

### 3.5 相关推荐

- 根据类型或关键字推荐 3 条资讯；如果是课程/活动相关，可推荐课程

### 3.6 评论区（可选）

- 简易评论模块：昵称、内容、发布
- 审核机制（配置开关）

---

## 4. 交互/动效

| 交互 | 说明 |
| --- | --- |
| Hero 滚动 | 视差效果，标题与背景分层 |
| 阅读进度条 | 跟随滚动显示阅读百分比 |
| 图片 | Hover 放大，点击开灯箱 |
| 分享 | 点击展开列表，带动效 |

---

## 5. 响应式策略

| 终端 | 处理 |
| --- | --- |
| PC | 正文+侧栏布局，Hero 全宽 |
| Pad | 侧栏下沉至正文后；字体略增大 |
| Mobile | Hero 改为 16:9 图，正文缩进减少，按钮改为底部浮条 |

---

## 6. 技术实现

| 领域 | 实现 |
| --- | --- |
| 数据 | `/news/detail`, `/news/list`, `/storeup/save` |
| 排版 | Markdown/富文本渲染（支持自定义组件） |
| 状态 | Pinia（收藏状态、阅读进度） |
| 动画 | GSAP、ScrollTrigger |

## 8. 验收标准

| 维度 | 标准 |
| --- | --- |
| 功能 | 正文渲染完整，分享/收藏可用 |
