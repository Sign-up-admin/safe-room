---# 明星教练详情页需求文档（Coach Detail v1.0）

> 版本：v1.0  
> 更新日期：2025-11-15  
> 适用路由：`/index/jianshenjiaolianDetail?id={coachId}`  
> 关联模块：`moduleKey = 'jianshenjiaolian'`

---

## 0. 设计关键词

沉浸人像 · 技能谱系 · 成果展示 · 预约闭环 · 信任背书

---

## 1. 页面逻辑结构

| 顺序 | 区块 | 目的 |
| --- | --- | --- |
| ① | 人像 Hero + 基本信息 | 突出教练形象与 USP，提供快速预约入口 |
| ② | 资格认证 / 奖项 | 展示证书、赛事成绩 |
| ③ | 技能谱系图 | 展示擅长领域、训练方法、成功案例 |
| ④ | 授课课程 | 列出可带课程，提供跳转 |
| ⑤ | 预约排期 | 当周时间表，支持立即预约 |
| ⑥ | 学员评价 & Before/After | 展示口碑与成果图 |
| ⑦ | FAQ & CTA | 解答常见问题，引导加入会员/客服 |

---

## 2. 视觉规范

- Hero：全屏半身照 + 金色线框，背景粒子沿人物轮廓流动
- 资格认证使用徽章式卡片
- 技能谱系采用环形图/雷达图
- Before/After 使用可拖动对比组件

---

## 3. 功能模块

### 3.1 Hero 信息区

| 元素 | 说明 |
| --- | --- |
| 头像/视频 | 支持短视频介绍，静音循环 |
| 文案 | 姓名、资历年限、擅长领域、代表成绩 |
| CTA | `预约私教`（带教练 ID）、`咨询顾问`、`收藏` |
| 数据 | 星级评分、服务人数、好评率 |

### 3.2 资格认证

- 列出国家/国际认证、赛事奖项
- 可点击查看证书大图

### 3.3 技能谱系

- 图表展示擅长项目（力量、HIIT、康复、青少年等）
- 支持“训练理念”富文本介绍

### 3.4 授课课程

- 列出当前教练负责的课程（来自 `/jianshenkecheng/list`）
- 显示课程类型、难度、剩余名额
- CTA：`预约该课程`、`查看课程详情`

### 3.5 预约排期

- 当周时间表 + 可选时间段
- 冲突检测：与用户已有私教预约对比
- 支持切换周视图

### 3.6 学员评价与成果

- 评价卡片：头像、星级、训练目标、结果描述
- Before/After 组件：上传前后对比图，支持拖动滑杆

### 3.7 FAQ & CTA

- 解答常见问题（课前准备、取消政策等）
- CTA：`加入会员`、`定制训练计划`、`AI 客服`

---

## 4. 交互/动效

| 交互 | 说明 |
| --- | --- |
| Hero 视差 | 人像与背景分层移动 |
| 资格认证 hover | 证书放大 + 光效 |
| 技能谱系切换 | Tab 切换时图表平滑过渡 |
| Before/After | 拖动滑杆对比图片 |
| 预约排期 | 选中时间段时展示确认卡片 |

---

## 5. 响应式策略

| 终端 | 处理 |
| --- | --- |
| PC | 双栏布局，Hero 全屏展示 |
| Pad | Hero 收缩为 16:9，排期以列表呈现 |
| Mobile | Hero 改为静态图，模块串行堆叠，排期使用横向滚动 |

---

## 6. 技术栈

| 领域 | 实现 |
| --- | --- |
| 媒体 | 使用 `IntersectionObserver` 控制视频播放 |
| 图表 | ECharts / Canvas 自绘雷达图 |
| 数据 | `getModuleService('jianshenjiaolian')` + 预约接口 |
| 状态 | Pinia（收藏、预约意向） |
| 动效 | GSAP, ScrollTrigger |

## 8. 验收标准

| 维度 | 标准 |
| --- | --- |
| 功能 | 课程、排期、评价完整，CTA 正常跳转 |
| 视觉 | 照片质量、动画曲线与首页一致 |
