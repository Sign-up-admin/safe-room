---# 会员卡总览页面需求文档（Membership Cards v1.0）

## 0. 设计关键词

卡片矩阵 · 权益可视化 · 价格梯度 · 对比交互 · CTA 闭环

---

## 1. 页面结构

| 顺序 | 区块 | 目的 |
| --- | --- | --- |
| ① | 卡种概览 Hero | 展示品牌宣言 + 会员价值 |
| ② | 会员卡矩阵 | 对比不同卡种权益、价格、适用场景 |
| ③ | 权益详情 | 图表/图标展示训练次数、私教折扣等权益 |
| ④ | 用户口碑 | 展示会员评价、续费率 |
| ⑤ | 常见问题 + CTA | 解答购买疑虑，引导购买/咨询 |

---

## 2. 视觉规范

- 卡片外观模拟实体卡：黑底 + 金色描边 + 霓虹边缘
- 选中卡片有发光流动效果
- 权益图标使用线性图标，保持科技黄映射
- 对比区域使用表格 + 热力高亮

---

## 3. 功能说明

### 3.1 会员卡矩阵

| 功能 | 说明 |
| --- | --- |
| 数据来源 | `/huiyuanka/list`（字段：卡种、价格、有效期、权益、推荐指数、热度排序） |
| 展示方式 | 3D卡片轮播 + 对比矩阵表，支持详情展开和多卡对比 |
| 权益标签 | 训练次数、私教折扣、设施权限、嘉宾待遇、预约优先级、积分累积 |
| CTA | `立即购买`（跳转支付流程）、`对比详情`、`咨询顾问`、`收藏卡片` |
| Hover动效 | 卡片3D上浮 + 金色描边流动 + 阴影扩散，显示完整权益清单 |

### 3.2 权益详情

- 可视化图表：使用ECharts展示不同卡种的权益对比（雷达图/柱状图/热力图）
- 权益解释：详细说明各项权益的具体含义和使用规则
- 对比工具：支持选择多张会员卡进行详细对比，突出差异点
- 价值计算：展示每种会员卡的性价比分析和推荐使用场景

### 3.3 用户口碑

- 数据统计：展示会员续费率、使用频率、满意度评分、平均训练提升
- 用户评价：精选真实用户评价，支持按卡种筛选和排序
- 成功案例：展示会员通过使用会员卡获得的健身成果和体验分享
- 社交证明：显示总会员数量、活跃会员占比、推荐转化率

### 3.4 FAQ + CTA

- 常见问题：会员卡种类说明、升级规则、转让政策、退费标准、权益使用规则
- 购买指南：详细的购买流程、使用须知、售后服务说明
- CTA：`在线咨询`、`立即购买`、`预约体验课`、`下载会员手册`

## 4. 交互与动效

| 交互 | 说明 |
| --- | --- |
| 卡片轮播 | 3D轮播效果，自动播放，鼠标悬停暂停，支持手势滑动 |
| 对比交互 | 多选卡片对比，差异高亮显示，支持拖拽排序 |
| CTA动效 | 渐变按钮，点击涟漪效果，购买按钮特殊强调 |
| 图表动画 | 切换卡片时图表平滑过渡，数据更新有加载动画 |

---

## 5. 响应式策略

| 终端 | 处理 |
| --- | --- |
| PC | 卡片矩阵三列展示，对比表全宽 |
| Pad | 卡片两列，对比表可横向滚动 |
| Mobile | 卡片单列滑动，对比表改为折叠列表 |

## 6. 技术实现

## 8. 验收标准

| 维度 | 标准 |
| --- | --- |
| 功能 | 会员卡信息准确展示、对比功能正常、CTA跳转正确、收藏功能可用 |

| 性能 | 轮播性能良好、图表加载 ≤ 1s、对比计算实时、首屏加载 ≤ 2s |

---

## 9. 备注

- 数据需支持后台配置权益描述、价格策略、活动标签
- 预留"限时活动"标签位，支持倒计时显示
- 会员卡数据需要支持A/B测试，不同用户展示不同推荐
- 收藏功能需要与用户账户关联，支持跨设备同步
- 图表数据需支持实时更新，展示最新的会员统计信息
