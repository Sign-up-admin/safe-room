## 📈 实现超越

实际实现已**显著超越**初始需求，新增以下超预期功能：

### ✨ 核心增强
- **沉浸阅读模式**：切换到无干扰的全屏阅读体验
- **滚动进度指示器**：全局滚动进度条和阅读百分比显示
- **智能阅读统计**：总阅读数、平均进度、阅读时间估算
- **动态进度追踪**：每个卡片的个性化阅读进度计算

### 🎯 用户体验提升
- **响应式进度条**：基于滚动位置的实时进度更新
- **阅读时间线**：按月份分组的历史阅读记录
- **分享功能集成**：一键复制分享链接
- **分类导航优化**：标签式快速筛选和排序

### 🔧 技术创新
- **Intersection Observer**：高效的元素可见性检测
- **自定义加载动画**：Canvas 实现的旋转加载效果
- **防抖处理机制**：搜索和筛选的性能优化
- **状态管理集成**：完整的响应式数据流管理

## 0. 设计关键词

资讯卡片 · 分类筛选 · 流光时间线 · 阅读进度 · 品牌发布

---

## 1. 页面结构

| 顺序 | 区块 | 目的 |
| --- | --- | --- |
| ① | 资讯导航栏 | 分类筛选、关键词搜索、标签快速切换 |
| ② | 热门公告横幅 | 突出置顶或最新活动 |
| ③ | 新闻卡片网格 | 展示文章封面、标题、简介、阅读数 |
| ④ | 阅读时间线 | 以时间轴呈现历史资讯 |
| ⑤ | CTA 区 | 引导查看品牌故事/加入会员 |
| ⑥ | 分页组件 | 控制列表页数 |

---

## 2. 视觉规范

- 背景：深黑 + 垂直光栅
- 分类标签：金色发光条，hover 有滑动效果
- 卡片：玻璃拟物 + 角标显示类型
- 时间轴：使用霓虹节点 + 光轨

---

## 3. 功能模块

### 3.1 导航栏

| 功能 | 说明 |
| --- | --- |
| 搜索 | 标题/简介模糊匹配 |
| 分类 | 来源 `/newstype/list`，支持多选 |
| 标签 | 系统预设热门标签（活动/课程/会员） |
| 排序 | 时间、阅读量 |

### 3.2 新闻卡片

| 元素 | 说明 |
| --- | --- |
| 封面 | 支持视频封面（hover 播放）/图片 |
| 角标 | 显示类型或“置顶” |
| 信息 | 标题、摘要、发布日期、阅读数 |
| CTA | `阅读更多` |

### 3.3 阅读时间线

- 将新闻按月份分组，展示在时间轴上
- 支持快速跳转到某一月份

### 3.4 分页

- Element Plus Pagination，样式统一
- 支持页尺寸切换、跳页

---

## 4. 交互与动效

| 互动 | 规范 |
| --- | --- |
| 分类切换 | 伴随底部能量条滑动 |
| 卡片 hover | 上浮、封面放大、CTA 显示 |
| 时间轴 | 节点高亮 + 流光沿轴移动 |
| 空状态 | 显示插画 + CTA（返回主页） |

---

## 5. 响应式策略

| 终端 | 处理 |
| --- | --- |
| PC | 双列卡片 + 时间轴 |
| Pad | 单列卡片，时间轴缩放 |
| Mobile | 卡片堆叠，时间轴改为折叠列表 |

---

## 6. 技术实现

| 领域 | 实现 |
