## 0. 设计关键词（Design Keywords）

- 紧迫感 · 价值传递 · 无缝续费
- 营造到期紧迫感的同时突出续费价值，提供便捷的无缝续费体验

---

## 1. 页面整体逻辑结构

| 顺序 | 区块 | 目的 |
| --- | --- | --- |
| ① | 到期倒计时面板 | 视觉化展示剩余时间，营造紧迫感 |
| ② | 会员权益回顾 | 展示当前权益和续费后权益对比 |
| ③ | 续费方案选择 | 提供多种续费时长和优惠方案 |
| ④ | 支付流程引导 | 安全便捷的支付体验 |
| ⑤ | 续费成功反馈 | 确认续费成功并展示新权益 |

## 2. 视觉风格规范

### 2.1 颜色

| 名称 | 颜色 | 用途 |
| --- | --- | --- |
| 科技黄 | `#FDD835` | 主 CTA、紧急续费按钮 |
| 深黑背景 | `#0A0A0A` | 页面背景 |
| 科技灰 | `#1A1A1A` | 卡片/分区背景 |
| 冷灰 | `#9EA1A6` | 辅助文字 |
| 紧急红 | `#F44336` | 到期警告，倒计时紧急状态 |
| 价值绿 | `#4CAF50` | 权益展示，续费价值强调 |
| 优惠橙 | `#FF9800` | 优惠价格，折扣信息 |

### 2.2 动效风格

- 倒计时动画：Canvas 圆环进度条动态变化
- 权益翻转：GSAP 实现权益卡片翻转对比效果
- 价格跳动：续费金额数字跳动动画
- Hover 行为：上浮 4px，发光增强 8%
- 页面滚动：各区块依次出现，营造流程感

### 2.3 卡片与组件规范

- 圆角：18px
- 背景：黑色玻璃拟物，`backdrop-filter: blur(8-12px)`
- 边框：根据到期紧急程度变化颜色（绿→黄→红渐变）
- 阴影：`0 20px 40px rgba(0, 0, 0, 0.45)`

## 3. 功能模块说明

### 3.1 到期倒计时可视化

| 功能 | 说明 | 数据来源 | 路由/接口 |
| --- | --- | --- | --- |
| 圆环倒计时 | 动态显示剩余天数百分比 | `huiyuanka.jieshushijian` 计算 | 实时前端计算 |
| 状态颜色变化 | 根据剩余天数变化颜色 | 前端状态判断逻辑 | 本地状态管理 |
| 到期信息展示 | 显示到期时间和会员信息 | `huiyuanka` 表 | `/huiyuanka/detail/{id}` |
| 紧急程度提示 | 不同时间段显示不同提示文案 | 配置化文案 | `/config/renewal/messages` |

### 3.2 权益对比展示

| 功能 | 说明 | 数据来源 | 路由/接口 |
| --- | --- | --- | --- |
| 当前权益展示 | 展示当前会员卡权益 | `huiyuanka.huiyuantequan` | `/huiyuanka/privileges` |
| 续费后权益预览 | 续费后将获得的权益 | `huiyuanka` 模板数据 | `/huiyuanka/preview/{type}` |
| 价值对比图表 | 续费价值可视化对比 | 前端图表计算 | ECharts 组件 |
| 过期风险提示 | 提醒即将失去的权益 | 业务规则配置 | `/config/renewal/risks` |

### 3.3 续费方案选择

| 功能 | 说明 | 数据来源 | 路由/接口 |
| --- | --- | --- | --- |
| 时长选择 | 1/3/6/12个月多种选择 | `huiyuanka.youxiaoqi` 配置 | `/huiyuanka/durations` |
| 优惠计算 | 自动计算续费优惠金额 | 优惠规则引擎 | `/pricing/calculate` |
| 批量续费 | 支持续费多张会员卡 | 会员卡列表选择 | `/huiyuanka/batch/renewal` |
| 自动续费设置 | 开启自动续费功能 | 用户偏好设置 | `/user/auto-renewal` |

### 3.4 支付流程集成

| 功能 | 说明 | 数据来源 | 路由/接口 |
| --- | --- | --- | --- |
| 支付方式选择 | 多种支付方式支持 | 支付配置 | `/payment/methods` |
| 订单预览 | 显示续费订单详情 | 订单生成接口 | `/order/preview` |
| 支付状态监控 | 实时显示支付进度 | WebSocket/轮询 | `/payment/status/{orderId}` |
| 支付结果反馈 | 成功/失败状态处理 | 支付回调 | `/payment/callback` |

## 4. 交互与动效规范

| 维度 | 规范 |
| --- | --- |
| 动画时长 | 0.3s ~ 0.55s |
| 缓动 | `cubic-bezier(0.4, 0, 0.2, 1)` 或 `GSAP Power3.easeInOut` |
| Hover | 发光增强 2%~10%，浮起 2~5px |
| 滚动触发 | 倒计时圆环旋转进入，权益卡片依次翻转 |
| 空状态 | 未到续费期显示鼓励使用插画 |
| 加载状态 | 支付中显示粒子动画进度条 |

## 9. 备注

- **依赖关系**：与会员卡系统、支付系统、提醒系统深度集成
- **业务规则**：续费可提前30天进行，过期后7天内可补续费
- **优惠策略**：续费折扣根据续费时长和历史消费计算
- **风险控制**：防止重复续费，续费后立即更新到期时间
