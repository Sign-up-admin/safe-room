# ADD_PAGES_KECHENGYUYUE

> **版本**：v1.0.0
> **更新日期**：2025-11-17
> **组件类型**：页面组件
> **适用框架**：Vue 3 + TypeScript + Element Plus
> **依赖组件**：ModuleFormPage, ModuleForm

---

## 📖 概述

### 组件描述

课程预约添加页面用于创建新的课程预约记录或编辑现有预约信息。该组件基于共享的模块表单组件，支持创建和编辑两种模式。

### 适用场景

- 创建课程预约记录
- 编辑预约信息
- 管理员预约管理

### 实现方式

使用共享的 `ModuleFormPage` 组件，支持通过路由参数 `id` 区分创建和编辑模式。
