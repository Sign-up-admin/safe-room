# LIST_PAGES_JIANSHENKECHENG

> **版本**：v1.0.0
> **更新日期**：2025-11-17
> **组件类型**：页面组件
> **适用框架**：Vue 3 + TypeScript + Element Plus
> **依赖组件**：ModuleListPage, ModuleList

---

## 📖 概述

### 组件描述

健身课程列表页面用于展示所有可用的健身课程，支持分页、筛选、排序等功能。该组件基于共享的模块列表组件，提供统一的课程浏览和管理体验。

### 适用场景

- 浏览健身课程
- 课程信息查询
- 预约课程入口
- 课程管理（管理员）

### 功能特性

- [x] **课程列表展示**：分页展示课程信息
- [x] **多条件筛选**：按课程类型、难度、教练等筛选
- [x] **排序功能**：按价格、热度、时间排序
- [x] **课程详情跳转**：点击查看课程详细信息
- [x] **预约功能入口**：直接跳转到课程预约页面
