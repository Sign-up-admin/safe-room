# 贡献者列表

感谢所有为 safe-room 项目做出贡献的开发者！

## 贡献者

本项目欢迎所有形式的贡献，包括但不限于：
- 代码贡献
- 文档改进
- Bug 报告
- 功能建议
- 代码审查

## 如何成为贡献者

1. Fork 本项目
2. 创建特性分支 (`git checkout -b feat/AmazingFeature`)
3. 提交更改 (`git commit -m 'Add some AmazingFeature'`)
4. 推送到分支 (`git push origin feat/AmazingFeature`)
5. 开启 Pull Request

详细的贡献指南请参考 [CONTRIBUTING.md](CONTRIBUTING.md)。
