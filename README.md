# Safe Room — 健身房管理与健身社区平台

“Safe Room” 是一个集 **健身房管理、健身社区互动、训练分享、健康生活推荐** 于一体的综合平台。
本平台旨在帮助用户更方便地管理健身计划、分享训练经验，并与社区成员交流。

---

## 🌟 功能特点

- 💪 **健身房管理系统**：包括会员管理、课程管理、训练计划安排等
- 🏋️ **训练记录与分享**：记录每日训练，发布训练心得
- 👥 **健身社区交流**：用户可发布帖子、评论、点赞，与其他运动爱好者互动
- 📅 **日程与打卡体系**：提醒用户保持训练习惯
- 📊 **健康数据展示**：如卡路里消耗、训练时长等（视具体功能可调整）
- 🔐 **安全与隐私保护**：用户数据加密存储，信息更安全
